This website is made up of lots of HTML elements. An element is defined by its start tag, end tag, and content sandwiched in between.

The "Hello world!" at the top is a header (<h1>)! It is the biggest header offered. You will see headers of other sizes around.

HTML Elements

To create lists, you can use

  1. <ol> to define an ordered (numbered) list, or
  2. <ul> to define an unordered list.

Within each list, you can then nest <li> elements to specify the actual list items. It is not a must to use the <ul> for unordered lists, but it may look different.

Let me quickly list out some other common elements!

  • Headers, from largest to smallest:
    1. <h1>
    2. <h2>
    3. <h3>
    4. <h4>
    5. <h5>
    6. <h6>
  • Paragraph: <p>
  • Line Break: <br> (closing tag </br> not needed)
  • Division/Section: <div>
  • Anchor (hyperlink): <a>
  • Image: <img>

Table elements

In HTML, a table is made up of a few parts:

  • <table> defines the table
  • <tr> defines a row
  • <td> defines a cell
  • <th> defines a header cell
